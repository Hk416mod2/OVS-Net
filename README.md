## Installiation

- Create a new conda environment
```
conda create -n univesselseg python=3.10
conda activate univesselseg
```

- Install UniVesselSeg
```
git clone https://github.com/Hk416mod2/UniVesselSeg
cd UniVesselSeg/
pip install -r requirement.txt
```
